## Método de Uso

O seguinte projeto, simula um ambiente de biblioteca, onde há clientes, livros, funcionários, entre outros. Há alguns elementos faltantes no projeto que poderiam ser melhorados, mas para apresentar os conceitos de POO já é o suficiente o que foi implementado.

Basicamente, há algumas coisas implicitas no projeto.
O arquivo.txt é utilizado para gravar "MidiaDigital" em um Livro, então é de lá que a função do Livro "setTextoMidiaDigital" utiliza para preencher o Livro selecionado.
Quando escolher a opção Salvar Livro, ou Carregar Livro do Menu, vai ser utilizado o arquivo "livro.ser", que irá conter um Objeto livro serializado/salvo, e que pode ser lido pelo "carregar".
